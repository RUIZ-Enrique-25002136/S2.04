# S2.04
## Comment marche la BDD des tenracs
Pour faire fonctionner le générateur données (et donc avoir les fichiers .sql et .csv) ilfaut insérer le nombre de tuples (nommé NB_XXXXX).
Il y a une possiblité d'envoyer les données générer vers une base de donneés Oracle en utilisant extension_oracle.py,le fichier nommé extension.py génere uniquement les fichiers .csv et .sql. Pour envoyer les données dans une base de données Oracle il faudra insérer l'identifiant et le mot de passe
